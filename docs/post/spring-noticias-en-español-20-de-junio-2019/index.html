Spring noticias en español, 20 de junio 2019

Bienvenidos otra vez la edición de junio de Spring Noticias en Español!!

Como siempre (!), envíame noticias de tus logros en @mkheck en Twitter, mark@losheckler.com, o mheckler@pivotal.io y los compartiré con la comunidad.

Para esta edición, aquí tienes unos artículos muy interesantes

  • Para continuar con la colleción de videos recientemente compartida conmigo de 4SoftwareDevelopers sobre diversos aspectos de Spring Framework y Spring Boot, aquí tienes unos más:

    • ¿Qué es un ORM?

    • ¿Qué es Spring Data?

    • Ejemplo Práctico con Spring Data

    • ¿Qué es Spring Boot?

    • Ejemplo práctico Spring Data REST

    • ¿Qué es Spring Roo?

    • Ejemplo Práctico Spring Roo

    • ¿Qué es Spring Batch?

¡Esperen que vienen muchos más!

  • Bienvenido este mes a def user='josdem' con muchas contribuciones de compartir con la comunidad, ¡vamonos!

    • Aquí está un artículo sobre Empezando con Spring Webflux que contiene una introducción excelente a programación reactiva con Spring WebFlux+Project Reactor.

    • Próximamente, un buen artículo sobre Spring Webflux el lado del Servidor que examina más el uso de WebFlux en el servidor/la nube.

    • Otro artículo, esta vez del perspectivo del client: Spring Webflux el lado del Cliente. ¡Me gusta mucho el WebClient y como él lo demuestra porque WebClient reemplaza RestTemplate y funciona bien con servicios de bloqueo y no bloqueo!

    • La seguridad es muy importante, y en este artículo José Luis discute y demuestra exactamente eso (en el contexto reactivo de WebFlux, por supuesto!): Seguridad con Spring Webflux

    • Y con más detalles de autenticación y autorización (¡y uso de MongoDB como repositorio de usuarios también!) es el último artículo de José Luis (de este mes, ya que vienen muchos más, ¡claro!): Seguridad con Spring Webflux y MongoDB

¡Gracias José Luis!

  • El Profesor P nos comparte este artículo muy interesante y completo sobre Estableciendo filtros en Spring. ¡Échale un vistazo!

  • Otro nuevo contribuyente este mes (¡bienvenido!) es Daniel Medina. Daniel nos da dos artículos interesantes:

    • Aquí tenemos material sobre como crear un y cambiar eventos entre aplicaciones Spring: Spring: Sistema de eventos

    • Y con algo un poco más caprichoso y divertido - ¡no debemos estar estoicos todo el tiempo! - Daniel comparte este artículo sobre cambiando la startup banner de Spring Boot: Tips Spring : BOOT banner ASCII de inicio

Y eso, queridos amigos, es todo por este mes…​¡wauw, que cosas buenas!

En resumen

Es todo por esta edición, pero hay mucho más en la comunidad, quizás en tus manos! Si te gustaría ver tus proyectos/noticias en este boletín y ayudar a tus compañeros, envíamelos! Mis datos de contacto son:

  • @mkheck en Twitter

  • mark@losheckler.com

  • mheckler@pivotal.io

Ayúdame a extender tu voz y compartir tus conocimientos con otras! Hasta la próxima edición…​ 👋😃
